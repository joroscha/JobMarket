## Best Project Example

### Generative Music Experiment

- Dissertation chapter on how generative AI affects musical creativity.
- Studied how a text-to-music system (`Udio`) changes the creative process during music composition.
- Used a randomized controlled experiment:
  - some participants had access to the AI tool during ideation
  - others did not
- Found that the AI-assisted condition led to lower perceived creativity scores, which was a surprising result.
- Also saw changes in the musical features of the compositions.
- I led the project end to end:
  - defined the question
  - designed the experiment
  - ran the study
  - built the web app used for evaluation
  - wrote up the results
- Why it matters for this role:
  - directly about evaluating generative music tools
  - not just technical performance, but whether they create real value for creators
